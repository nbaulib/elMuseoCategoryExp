# El Museo Press Tool

A simple tool for exporting press contacts from an Excel spreadsheet based on categories.

## What it does

This app takes an Excel file with contact categories and:

- Splits multiple categories into separate rows
- Creates a CSV file containing the expanded spreadsheet
- Exports separate CSV files for each category

## How to use
1. Go to https://elmuseo-press-tool.streamlit.app/
2. Upload Excel spreadsheet
3. Download the exported CSV files

## Built With

- Python
- Pandas
- Streamlit
- OpenPyXL
